**Maindak** (the Hindi word for Frog) is a futuristic, AI-driven health assistant designed to simplify medical complexities. Far more than a standard search engine, it represents a sophisticated fusion of professional medical data scraping and an immersive, interactive user experience.

---

## 🌟 Spotlight Features

### 🤒 **Smart Symptom Engine**
Utilizes **real-time extraction** from verified medical databases to provide accurate symptoms, treatments, and essential precautions.

### 💊 **Pharma Intelligence**
Delivers instant insights into **Over-the-Counter (OTC)** medication names and detailed salt compositions.

### 📄 **Professional Medical Reports**
A one-click solution to transform comprehensive health analyses into **high-quality, downloadable PDF documents**.

### 🌐 **Linguistic Fluidity**
Seamlessly integrated with **Google Translate**, ensuring health accessibility for users in their native languages.

### 🌙 **Ethereal UI/UX**
Features a premium aesthetic with **Liquid Mesh Gradients**, **Glassmorphism cards**, and an optimized **Eye-Care dark mode**.

### 🐸 **Interactive Mascot**
Meet **Froggy**, your personal guide. Froggy brings the application to life through **reactive hover effects** and dynamic animations.

---

## 🚀 The Tech Ecosystem

### **The Frontend (The Face)**
*   **Vite + React:** Leveraged for industry-leading performance and a fluid **Single Page Application (SPA)** feel.
*   **Glassmorphism:** A modern design language using **frosted glass effects** for a sophisticated, professional interface.
*   **html2pdf.js:** Seamlessly bridges web components and document generation for **pixel-perfect PDF exports**.

### **The Backend (The Brain)**
*   **Python Flask:** A robust yet lightweight **micro-framework** designed to manage API requests and logic efficiently.
*   **BeautifulSoup4:** The **"eyes" of Maindak**, navigating medical repositories to extract relevant data.
*   **Regex Intelligence:** Advanced **data sanitization** to filter out noise, providing only high-value medical insights.

## 🛠️ Quick Installation Guide

### **1. Clone the Repository**
```bash
git clone https://github.com/vivekjoshi2006/Maindak-The-Health-AI.git
cd Maindak_Health_AI
```

### **2. Initialize the Backend**
```bash
cd backend
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### **3. Launch the Frontend**
```bash
cd frontend
npm install
npm run dev
```
